# 软换行

没有前面有两个或多个空格或反斜杠的常规换行符（不在行内代码或 HTML 标签中）被解析为软换行。 （软换行可以在 HTML 中被解析为行结尾或空格。结果在浏览器中将是相同的。在此处的示例中，将被解析为行结尾。）
示例 647

foo
baz


<p>foo
baz</p>

删除行尾和下一行开头的空格：
示例 648

foo 
 baz


<p>foo
baz</p>

符合标准的解析器可以在 HTML 中渲染软换行符，作为换行符或空格。
渲染器还可以提供将软换行渲染为强制换行的选项。
